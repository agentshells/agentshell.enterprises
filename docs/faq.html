Common questions about Agent Shell Enterprises Ltd.

Company Information

  • Agent Shell Enterprises Ltd is a Finnish company providing secure, EU-hosted infrastructure for agentic workflows and autonomous developer tools. The company focuses on long-term infrastructure provision for agentic development tools, with all operations conducted within European Union data centers.

  • The Business ID is FI35734143. This uniquely identifies the company in Finland's Business Information System (YTJ).

  • Agent Shell Enterprises Ltd is registered in Tampere, Finland. The registered address is Keskustori 7 A 14, 33100 Tampere, Finland.

  • Agent Shell Enterprises Ltd was founded on 2025-10-20 and officially registered on 2025-10-31.

  • Agent Shell will launch a separate startup to develop the AgentHub agentic development platform. AgentHub product IP will be transferred and developed as a standalone product company; Agent Shell Enterprises remains focused on core infrastructure provision.
