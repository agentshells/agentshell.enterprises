## Specifications

### Logo
- Format: SVG preferred, PNG for downloads
- Should work at 32px height in header
- Brand colors: graphite (#111111), teal (#008080)

### Social Preview Image
- Dimensions: 1200x630px
- Format: JPEG or PNG
- Should include company name and branding
- Used for Open Graph and Twitter Card previews

### Favicon
- Format: ICO
- Standard sizes: 16x16, 32x32, 48x48
